# UPI QR Code Generator (Python)

This project generates UPI payment QR codes for PhonePe, Paytm, and Google Pay using Python.

## Features
- Accepts UPI ID from user
- Generates UPI payment QR codes
- Saves QR codes as PNG files
- Opens QR images automatically

## Requirements
- Python 3.x
- qrcode library
- pillow library

## Installation
```bash
pip install qrcode[pil]
```

## Run the Project
```bash
python upi_qr.py
```

## Output
- phonepay_qr.png
- paytm_qr.png
- gpay_qr.png
